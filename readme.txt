=== Agent Photo ===

Agent Photo enhances the Gutenberg image block with an AI-powered assistant that generates descriptive alt text, a legend, and creative title options. When you press the "Process Image" button in the block settings sidebar, the image is processed via OpenAI's API.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/agent-photo` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to Settings > Agent Photo to enter your OpenAI API key.
